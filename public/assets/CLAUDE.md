# public/assets/

Runtime art (`tiles/`, `objects/`, `characters/`) and audio (`audio/`). Served as-is by
Vite; loaded via Phaser's loader (see `src/assetConfig.ts`).

- Provenance and licenses for every file are recorded in `CREDITS.md` (repo root) — keep it
  in sync when adding assets.
- Not all game art lives here: Avatars, Village Buildings and Wildlife are original
  code-drawn sprites (`src/avatars.ts`, `src/ui/icons.ts`, `src/mobSprites.ts`), no PNGs.

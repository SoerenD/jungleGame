# public/assets/tiles/

Terrain tilesheets sampled by the map renderer: `terrain.png` (16×16 overworld tiles),
`water-frames.png` (animated water), `mire-tiles.png` (Sunken Mire Realm palette).

- Tile indices here must line up with the generator (`tools/generate-map.ts`) and the map JSON
  in `public/map/`. Changing tile order means regenerating the map.
